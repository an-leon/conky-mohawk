# conky-mohawk

| Description | Side Bar |
| - | - |
| Bash file for a conky side bar. The classic Conky Seamod has plenty of fun widgets, but I wanted something easier to read at a glance. This script strips out or repairs many of the components, adjusts the color scheme, and fine-tunes the transparency for a simple, clean look. </br></br></br> **How to use:**  This script was written for use in [Solus](https://getsol.us/home/). To get the ethernet & wifi detection to work properly for a different operating system, you may need to change "eno1" to "eth0" on lines 99,101,102 and change "wlp3s0" to "wlan0" on lines 105,107,108.</br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br></br>| <img src="https://github.com/MohawkDev/conky-mohawk/blob/assets/ConkySideBar.png" alt="New & Improved Conky Side Bar" width = "1400" height = "900"> |
